# Bizmo — ビジネスモデル・シミュレーションエンジン

Bizmo は、ビジネスモデルを **事業計画の数値表** として表現しながら、事業を構成する要素と行動を **ノードと有向グラフ** で直感的に可視化し、シミュレーションを行うためのWebアプリケーションです。

シミュレーションを通じて、事業計画の **上振れ・下振れのシナリオ分析** や、売上・利益に大きく影響する **経営上の重要パラメータの特定** を可能にすることを目的としています。

## 何ができるか

-   **ビジネスモデルの構造化**: 自社・仕入先・顧客・外部環境を構成要素（ノード）として定義し、それらの間の関係を有向グラフで表現
-   **事業計画のシミュレーション**: 定義した構造に対して期間ごと（月次）の数値計算を実行し、推移表として結果を確認
-   **複式簿記の自動仕訳**: 金額の変動が自動的にB/S・P/Lの仕訳に反映
-   **コホート分析・ファネル分析**: Excel では実現が難しい、経過期間ベースの顧客維持率や、段階的なコンバージョン率の計算を構造的に定義・シミュレート可能
-   **確率分布パラメータ**: 固定値だけでなく、正規分布・一様分布・三角分布からのサンプリングによる確率的シミュレーション
-   **独自の数式エンジン**: `bizio0 * bizio1 + hyper0` のような数式で計算ロジックを柔軟に定義

## コンセプト

事業計画は本来、以下の2つの側面を持ちます：

1. **構造**: 誰が・何を・誰に対して行うのか（ビジネスモデル）
2. **数値**: いつ・いくら・何件（事業計画書）

従来のスプレッドシートでは (2) は得意ですが、(1) の構造を表現することが困難です。Bizmo はこの両方を統合し、構造を定義すれば数値が計算される仕組みを提供します。

### 事業の構成要素（BizComponent）

| 要素         | 説明                     | 例                                 |
| ------------ | ------------------------ | ---------------------------------- |
| **外部環境** | 事業の外側にある前提条件 | 為替レート、商圏人口               |
| **自社**     | 組織構造・リソース       | 従業員、製品、材料、固定資産、会計 |
| **顧客**     | ユーザーライフサイクル   | ファネル、コホート、セグメント     |
| **協力者**   | サプライチェーン上の他社 | 仕入先、販売代理店                 |

### 事業活動（BizAction）

構成要素間のデータフロー（有向グラフ）と計算ロジックを定義します。タイムライン上で優先度順に実行されます。

-   **HR テンプレート**: 採用・退職・異動・給与支払い
-   **会計テンプレート**: B/S 初期値設定

### Excel では難しい計算

| 機能                 | 説明                                                       |
| -------------------- | ---------------------------------------------------------- |
| **コホート分析**     | サインアップ後の経過月数に基づく継続率のモデル化           |
| **ファネル分析**     | マーケット → リーチ → 購入 → サブスク の段階的変換率       |
| **DAG 依存解決**     | BizIO 間の依存関係をトポロジカルソートで自動解決           |
| **複式簿記連動**     | 数値の変動が仕訳に自動反映、B/S・P/L の整合性を維持        |
| **確率的パラメータ** | 固定値と確率分布を混在させたモンテカルロ的シミュレーション |

## 画面構成

-   **キャンバス**: ノードと有向グラフでビジネスモデルの構造を可視化。ドラッグ&ドロップで配置
-   **時系列データ（推移表）**: 全構成要素の値を期間別に一覧表示。親子関係の折りたたみ対応
-   **データ構造**: ツリー/リストで構成要素を閲覧・編集
-   **BizAction**: 事業活動の一覧・新規作成・Relations/Processors/Priorities の編集
-   **パラメータリスト**: ハイパーパラメータの管理・確率分布の設定
